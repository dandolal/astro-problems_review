Задачи

На нашем сайте собраны или по крайней мере будут собраны все актуальные на данный момент задачи по олимпиадной астрономии.

Авторы

У каждой задачи, как известно, есть автор. На нашем сайте возможно посмотреть все задачи авторства конкретного человека.

Темы

Для всех задач в нашей базе указаны связанные с ней разделы астрономии. Конечно, всего в несколько касаний можно получить подборку задач по интересующией вас теме.

Поиск

И, конечно, мы реализовали поиск задач по словам из условия или решения. Более того, зарегистрированные пользователи могут сами добавлять задачи в базу.
